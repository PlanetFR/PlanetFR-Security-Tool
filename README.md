# PlanetFR Security Tool 🚀

A lightweight, open-source batch script wrapper for **Windows Defender** and **System File Checker (SFC)**.

## 🛠 Features
- **Quick & Full Scans:** Trigger Windows Defender via command line.
- **Targeted Scanning:** Easily scan your Downloads folder.
- **System Repair:** Run SFC Scans to fix corrupted Windows files.
- **Live Spinner:** Visual feedback so you know the scan is active.
- **Logging:** Automatically saves scan history to `PlanetFR_Log.txt`.
- **Custom UI:** Includes a theme toggle (Cyan/Green).

## ⚠️ Important
This tool **must be run as Administrator** to access the Windows Defender engine and system repair tools.
